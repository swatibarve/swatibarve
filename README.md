## Featured Projects

### Global Electronics Retailer — Power BI + ETL Pipeline
> Sales analytics for a global electronics retailer
> spanning 11 countries and 62,884 transactions

**What I built:**
- 5-table star schema data model in Power BI
- Multi-currency revenue consolidation using DAX LOOKUPVALUE
- End-to-end Python ETL pipeline (Extract → Transform → Load)
- 4 interactive dashboard pages with time intelligence measures
- 15 SQL queries covering YoY growth, window functions, CTEs

**Key findings:**
- Revenue peaked in 2019, declined 48% by 2021 (COVID impact)
- Computers = 35% of total revenue at $19.1M
- Delivery time improved from 7.3 days → 3.8 days (48% faster)
- Canada stores outperform USA on revenue per sq.m.

**Tools:** Power BI · DAX · Python · pandas · SQL · Numypy ·SQLite

### Python ETL Data Pipeline
> Automated data pipeline for Global Electronics dataset

**What I built:**
- Extract stage: reads 5 CSV sources, validates data quality
- Transform stage: fixes types, calculates USD revenue using
  exchange rates, and delivery metrics
- Load stage: writes to the SQLite database and cleans CSVs
- Full pipeline runs in under 10 seconds

**Tools:** Python · pandas · SQLite · SQLAlchemy
